# BoxFusion-Grads

## Project description
A platform where people can make and manage clubs with other people in the same neighborhood 
 
# Install
```bash
npm install
```

# Run
```bash
npx turbo run dev
```

# Build
```bash
npx turbo run build
```

## Stake Holders 
 - Neighbors 
 - Club Members
 - Neighborhood presidents

## Functional Requirements - Ordered By Priority 

### Club Management : 
1. Create a club (name, description, category, location)

2. Add club images/logo

3. Club rules or guidelines

4. Manage membership (approve/deny requests)

5. Edit club information

6. Delete or archive club

### Search & Discovery

1. Search clubs by:

- Location

- Category

- Name

- Popularity

2. Recommended clubs (based on interests & location)

### Location & Neighborhood System

1. User location detection (GPS or manual address entry)

2. Neighborhood grouping logic

3. Radius-based club visibility (e.g., show clubs within 10km)

### Club Events:

1. Create events for club members

2. Title, description, date, time, location

3. RSVP system (going / interested / not going)

4. Reminder notifications

5. Event chat or discussion

### Messaging & Communication

1. Club-wide chat

2. Private messaging between users

3. Group chat per event

4. Media sharing (images, files)



## Non-Functional RequireMents

###  Security & Moderation

1. Report posts/users/clubs

2. Admin and mod tools

3. Content filtering

4. Block or mute users

6. Auto-flag inappropriate content

### User Accounts & Authentication

1. Sign up / login (email, phone, or social login)

2. Profile creation (name, bio, location, interests)

3. Account verification (email/SMS)

4. Password reset

### Posts & Announcements

1. Create post (text, images, events)

2. Comments & replies

3. Likes/reactions

4. Content moderation tools for admin

### Notifications System

1. Club announcements

2. Event reminders

3. Membership requests

4. New posts or comments

5. Push notifications (mobile/web)
